# Security Policy

## Supported Versions

Currently, only the main branch of this repository is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| Old     | :x:                |

## Reporting a Vulnerability

We take the security of our open-source projects seriously. If you discover a security vulnerability within this project, please follow these steps:

1. **Do not open a public issue.** This ensures that the vulnerability isn't exploited before we can fix it.
2. Email the maintainer directly or use GitHub's private vulnerability reporting feature (if enabled on the repository).
3. Include a detailed description of the vulnerability, the steps to reproduce it, and the potential impact.

### Response Timeline

- You should receive an acknowledgment within 48 hours.
- We aim to provide a timeline for a fix within 7 days.
- Once the vulnerability is resolved, we will publish a security advisory and credit you for the discovery (unless you prefer to remain anonymous).

Thank you for helping keep this project secure!
